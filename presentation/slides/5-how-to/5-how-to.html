How to use jQuery?

  • Add it to your project via npm or yarn
    • npm i jquery
    yarn add jquery
  • Include it as script in your HTML before any JS that uses it
    • <script src="node_modules/jquery/dist/jquery.min.js"/>
  • Or directly include it via a CDN
    • <script src="https://code.jquery.com/jquery-3.6.0.min.js"/>
